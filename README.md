# handwritten-to-data

Project workspace for converting handwritten inputs into structured data.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Structure

- `data/` - source datasets and local data files
- `notebooks/` - Jupyter notebooks for experiments
- `src/` - reusable Python source code
- `docs/` - project documentation
- `outputs/` - generated outputs and reports
- `models/` - trained or downloaded model files
